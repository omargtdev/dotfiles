# My Dotfiles
This is my dotfiles I've collected of distinct repos and configs along this year.
I leave the configs with their respective repositories I've found and used.

## Why are there so many .configs dirs?
I'm using [stow](https://alexpearce.me/2016/02/managing-dotfiles-with-stow/) to manage them.

## Some configs

| polybar-theme  | https://github.com/adi1090x/polybar-themes |
